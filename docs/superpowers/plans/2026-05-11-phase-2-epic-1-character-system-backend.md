# Phase 2 Epic 1 — Character system backend implementation plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the backend half of the Phase 2 Epic 1 spec (`docs/superpowers/specs/2026-05-11-phase-2-epic-1-character-system-design.md`): finish data-ingest cleanup, land schema additions, build the derivation function, change the character ↔ participant seam, add `SwapKit` and `Respite` intents, ship the `/characters` HTTP routes (with invite-code flow), and sweep the docs. Frontend (wizard + sheet) gets a follow-on plan.

**Architecture:** Characters are user-owned JSON blobs in D1. A pure derivation function in `packages/rules` (`deriveCharacterRuntime`) computes resolved runtime values from a draft-tolerant `CharacterSchema` plus bundled static data. The lobby roster carries thin PC placeholders between encounters and materializes full participants at `StartEncounter`. Runtime state (`currentStamina`, `recoveries.current`) persists between encounters; the structural fields are re-derived each `StartEncounter` so level-ups propagate automatically.

**Tech Stack:** TypeScript strict, Zod, Drizzle ORM, Cloudflare Workers + Durable Objects, D1, Hono, pnpm workspaces, Vitest.

**Pre-flight reading:** Engineers MUST read the spec before starting any phase. It is the authoritative source for behavior and edge cases. This plan focuses on execution order, file inventory, exact code, and verification gates.

  const BENEFIT_RE = /^\*\*Benefit:\*\*\s*(.+)$/m;

  // AFTER — tolerates both **Benefit:** inline and ### Benefit / **Benefit** heading variants
  const BENEFIT_INLINE_RE = /^\*\*Benefit\*\*[:.]?\s*(.+)$/m;
  const BENEFIT_HEADING_RE = /^(?:###|\*\*)\s*Benefit\s*\*?\*?\s*\n+([\s\S]+?)(?=\n+(?:###|\*\*\s*Drawback|\*\*\s*$))/m;

  function extractBenefit(body: string): string {
    const inline = body.match(BENEFIT_INLINE_RE);
    if (inline) return inline[1].trim();
    const heading = body.match(BENEFIT_HEADING_RE);
    if (heading) return heading[1].trim();
    throw new Error('missing Benefit field');
  }

- [ ] **B3: Implement `CompleteCharacterSchema`**

  Edit `packages/shared/src/character.ts`. Append (after the existing `CharacterSchema` export, before `CharacterResponseSchema`):

  ```ts
  // ── Submission-validity gate ──────────────────────────────────────────────
  //
  // CharacterSchema is draft-tolerant — every field nullable / defaulted.
  // CompleteCharacterSchema layers refinements that a fully-built character
  // must satisfy. Used by:
  //   - the wizard's Submit button enable state
  //   - POST /characters when both campaignCode and data are present
  //   - applySubmitCharacter authority check
  //
  // Per-career skill-count / language-count refinements require static-data
  // lookup and live in the wizard / reducer rather than the schema; this
  // schema enforces structural completeness only.

  export const CompleteCharacterSchema = CharacterSchema.refine(
    (c) => c.ancestryId !== null,
    { message: 'ancestry required', path: ['ancestryId'] },
  )
    .refine((c) => c.culture.environment !== null, {
      message: 'culture.environment required',
      path: ['culture', 'environment'],
    })
    .refine((c) => c.culture.organization !== null, {
      message: 'culture.organization required',
      path: ['culture', 'organization'],
    })
    .refine((c) => c.culture.upbringing !== null, {
      message: 'culture.upbringing required',
      path: ['culture', 'upbringing'],
    })
    .refine((c) => c.culture.environmentSkill !== null, {
      message: 'culture.environmentSkill required',
      path: ['culture', 'environmentSkill'],
    })
    .refine((c) => c.culture.organizationSkill !== null, {
      message: 'culture.organizationSkill required',
      path: ['culture', 'organizationSkill'],
    })
    .refine((c) => c.culture.upbringingSkill !== null, {
      message: 'culture.upbringingSkill required',
      path: ['culture', 'upbringingSkill'],
    })
    .refine((c) => c.culture.language !== null, {
      message: 'culture.language required',
      path: ['culture', 'language'],
    })
    .refine((c) => c.careerId !== null, {
      message: 'career required',
      path: ['careerId'],
    })
    .refine((c) => c.careerChoices.incitingIncidentId !== null, {
      message: 'inciting incident required',
      path: ['careerChoices', 'incitingIncidentId'],
    })
    .refine((c) => c.classId !== null, {
      message: 'class required',
      path: ['classId'],
    })
    .refine((c) => c.characteristicArray !== null, {
      message: 'characteristic array required',
      path: ['characteristicArray'],
    })
    .refine(
      (c) => {
        for (let lvl = 1; lvl <= c.level; lvl++) {
          if (!c.levelChoices[String(lvl)]) return false;
        }
        return true;
      },
      { message: 'levelChoices must cover every level up to `level`', path: ['levelChoices'] },
    );

  export type CompleteCharacter = z.infer<typeof CompleteCharacterSchema>;
  ```

  Note: the `kitId !== null if the class uses a kit`, `subclassId !== null if the class has subclasses`, and `careerChoices.skills.length matches careerCount` checks all require static-data lookup and live in the wizard's submit-gate logic + reducer authority check, not the schema. The schema enforces what it can without external data.

- [ ] **C4: Implement `deriveCharacterRuntime`**

  Fill in `packages/rules/src/derive-character-runtime.ts`:

  ```ts
  import type { Character, Characteristics } from '@ironyard/shared';
  import type { StaticDataBundle } from './static-data';
  import { requireCanon } from './require-canon';

  export type CharacterRuntime = {
    characteristics: Characteristics;
    maxStamina: number;
    recoveriesMax: number;
    recoveryValue: number;
    heroicResource: { name: string; max: number | null; floor: number };
    abilityIds: string[];
    skills: string[];
    languages: string[];
    immunities: Array<{ kind: string; value: number }>;
    weaknesses: Array<{ kind: string; value: number }>;
    speed: number;
    size: string;
    stability: number;
    freeStrikeDamage: number;
  };

  const ZERO_CHARACTERISTICS: Characteristics = {
    might: 0,
    agility: 0,
    reason: 0,
    intuition: 0,
    presence: 0,
  };

  export function deriveCharacterRuntime(
    character: Character,
    staticData: StaticDataBundle,
  ): CharacterRuntime {
    const cls = character.classId ? staticData.classes.get(character.classId) : null;
    const kit = character.kitId ? staticData.kits.get(character.kitId) : null;
    const ancestry = character.ancestryId
      ? staticData.ancestries.get(character.ancestryId)
      : null;

    // Characteristics: pair the chosen array with the class characteristic-slot order.
    const characteristics = deriveCharacteristics(character, cls);

    // Stamina: starting + (level - 1) * perLevel + characteristicMultiplier * relevantChar.
    const maxStamina = requireCanon('character-derivation.max-stamina')
      ? deriveMaxStamina(character, cls, characteristics) + (kit?.staminaBonus ?? 0)
      : 0;

    const recoveriesMax = requireCanon('character-derivation.recoveries')
      ? cls?.recoveriesPerLevel ?? 0
      : 0;

    const recoveryValue = requireCanon('character-derivation.recovery-value')
      ? Math.floor(maxStamina / 3)
      : 0;

    const heroicResource = cls?.heroicResource ?? { name: 'unknown', max: null, floor: 0 };

    const abilityIds = collectAbilityIds(character);
    const skills = collectSkills(character);
    const languages = collectLanguages(character);

    const immunities: CharacterRuntime['immunities'] = [];
    const weaknesses: CharacterRuntime['weaknesses'] = [];

    const speed = ancestry?.speed ?? 5;
    const size = ancestry?.size ?? '1M';
    const stability = (ancestry?.stability ?? 0) + (kit?.stabilityBonus ?? 0);
    const freeStrikeDamage = (kit?.meleeDamageBonus ?? 0) + 2; // canon: free-strike base 2

    return {
      characteristics,
      maxStamina,
      recoveriesMax,
      recoveryValue,
      heroicResource: {
        name: heroicResource.name,
        max: heroicResource.max ?? null,
        floor: heroicResource.floor,
      },
      abilityIds,
      skills,
      languages,
      immunities,
      weaknesses,
      speed,
      size,
      stability,
      freeStrikeDamage,
    };
  }

  function deriveCharacteristics(
    character: Character,
    cls: { characteristicSlots?: string[] } | null | undefined,
  ): Characteristics {
    if (!requireCanon('character-derivation.characteristics')) return ZERO_CHARACTERISTICS;
    if (!character.characteristicArray || !cls?.characteristicSlots) return ZERO_CHARACTERISTICS;

    const out: Characteristics = { ...ZERO_CHARACTERISTICS };
    cls.characteristicSlots.forEach((slot, idx) => {
      const value = character.characteristicArray![idx];
      if (typeof value === 'number') {
        (out as Record<string, number>)[slot] = value;
      }
    });
    return out;
  }

  function deriveMaxStamina(
    character: Character,
    cls:
      | {
          startingStamina?: number;
          staminaPerLevel?: number;
          staminaCharacteristic?: keyof Characteristics;
          staminaCharacteristicMultiplier?: number;
        }
      | null
      | undefined,
    characteristics: Characteristics,
  ): number {
    if (!cls) return 0;
    const base = cls.startingStamina ?? 0;
    const perLevel = (cls.staminaPerLevel ?? 0) * Math.max(0, character.level - 1);
    const charMod = cls.staminaCharacteristic
      ? (characteristics[cls.staminaCharacteristic] ?? 0) *
        (cls.staminaCharacteristicMultiplier ?? 0)
      : 0;
    return base + perLevel + charMod;
  }

  function collectAbilityIds(character: Character): string[] {
    const out: string[] = [];
    for (const lvl of Object.keys(character.levelChoices)) {
      const choices = character.levelChoices[lvl];
      out.push(...choices.abilityIds);
      out.push(...choices.subclassAbilityIds);
    }
    return out;
  }

  function collectSkills(character: Character): string[] {
    const out: string[] = [];
    if (character.culture.environmentSkill) out.push(character.culture.environmentSkill);
    if (character.culture.organizationSkill) out.push(character.culture.organizationSkill);
    if (character.culture.upbringingSkill) out.push(character.culture.upbringingSkill);
    out.push(...character.careerChoices.skills);
    for (const lvl of Object.keys(character.levelChoices)) {
      const sk = character.levelChoices[lvl].skillId;
      if (sk) out.push(sk);
    }
    return [...new Set(out)];
  }

  function collectLanguages(character: Character): string[] {
    const out: string[] = [];
    if (character.culture.language) out.push(character.culture.language);
    out.push(...character.careerChoices.languages);
    return [...new Set(out)];
  }
  ```

  import type { CampaignState, IntentResult, StampedIntent } from '../types';

  export function applySwapKit(state: CampaignState, intent: StampedIntent): IntentResult {
    const payload = SwapKitPayloadSchema.parse(intent.payload);

    // Reject mid-encounter.
    if (state.encounter !== null) {
      return {
        state,
        derived: [],
        log: [],
        errors: [{ code: 'in_encounter', message: 'cannot swap kits during an active encounter' }],

      owner?.kind === 'pc-placeholder' && owner.ownerId === intent.actor.userId;
    const isActiveDirector = intent.actor.userId === state.activeDirectorId;

      return {
        state,
        derived: [],
        log: [],
        errors: [{ code: 'permission_denied', message: 'owner or active director only' }],
      };

    // Side-effect: D1 mutation happens in the DO outside the reducer.
    return {
      state,
      derived: [],
      log: [

          kind: 'note',
          message: `Swapped kit for ${payload.characterId} to ${payload.newKitId}.`,
        },
      ],
    };
  }

  import { Hono } from 'hono';
  import { eq, and, or } from 'drizzle-orm';
  import { ulid } from 'ulid';
  import {
    CreateCharacterRequestSchema,

  import { authMiddleware } from '../auth/middleware';
  import { characters, campaigns, campaignMemberships } from '../db/schema';

  export const characterRoutes = new Hono<AppEnv>()
    .use('*', authMiddleware)
    .get('/', async (c) => {
      const userId = c.var.userId;
      const db = c.var.db;
      const rows = await db
        .select()
        .from(characters)
        .where(eq(characters.ownerId, userId))
        .all();
      return c.json(
        rows.map((row) => ({
          id: row.id,
          ownerId: row.ownerId,
          name: row.name,
          data: CharacterSchema.parse(JSON.parse(row.data)),
          createdAt: row.createdAt,
          updatedAt: row.updatedAt,
        })),
      );
    })
    .get('/:id', async (c) => {
      const userId = c.var.userId;
      const id = c.req.param('id');
      const db = c.var.db;
      const row = await db.select().from(characters).where(eq(characters.id, id)).get();
      if (!row) return c.json({ error: 'not_found' }, 404);

      // Owner gets through unconditionally.
      if (row.ownerId === userId) {
        return c.json({
          id: row.id,
          ownerId: row.ownerId,
          name: row.name,
          data: CharacterSchema.parse(JSON.parse(row.data)),
          createdAt: row.createdAt,
          updatedAt: row.updatedAt,
        });
      }

      // Otherwise: visible to campaign members if the character has a campaignId.
      const data = CharacterSchema.parse(JSON.parse(row.data));
      if (data.campaignId) {
        const membership = await db
          .select()
          .from(campaignMemberships)
          .where(
            and(
              eq(campaignMemberships.campaignId, data.campaignId),
              eq(campaignMemberships.userId, userId),
            ),
          )
          .get();
        if (membership) {
          return c.json({
            id: row.id,
            ownerId: row.ownerId,
            name: row.name,
            data,
            createdAt: row.createdAt,
            updatedAt: row.updatedAt,
          });
        }
      }
      return c.json({ error: 'forbidden' }, 403);
    })
    .post('/', async (c) => {
      const userId = c.var.userId;
      const body = await c.req.json();
      const parsed = CreateCharacterRequestSchema.safeParse(body);
      if (!parsed.success) return c.json({ error: 'invalid_request', issues: parsed.error.issues }, 400);
      const db = c.var.db;

      // Resolve campaign if campaignCode present.
      let campaignId: string | null = null;
      if (parsed.data.campaignCode) {
        const campaign = await db
          .select()
          .from(campaigns)
          .where(eq(campaigns.inviteCode, parsed.data.campaignCode))
          .get();
        if (!campaign) return c.json({ error: 'campaign_not_found' }, 404);
        campaignId = campaign.id;

        // Idempotent membership insert.
        const existing = await db
          .select()
          .from(campaignMemberships)
          .where(
            and(
              eq(campaignMemberships.campaignId, campaignId),
              eq(campaignMemberships.userId, userId),
            ),
          )
          .get();
        if (!existing) {
          await db
            .insert(campaignMemberships)
            .values({
              campaignId,
              userId,
              isDirector: 0,
              joinedAt: Date.now(),
            })
            .run();
        }
      }

      // Build the initial character blob.
      const initialData: CharacterSchema = parsed.data.data
        ? { ...parsed.data.data, campaignId }
        : CharacterSchema.parse({ campaignId });

      const id = ulid();
      const now = Date.now();
      await db
        .insert(characters)
        .values({
          id,
          ownerId: userId,
          name: parsed.data.name,
          data: JSON.stringify(initialData),
          createdAt: now,
          updatedAt: now,
        })
        .run();

      // Auto-submit if conditions are met.
      let autoSubmitted = false;
      if (
        parsed.data.campaignCode &&
        parsed.data.data &&
        CompleteCharacterSchema.safeParse(parsed.data.data).success
      ) {
        // Dispatch SubmitCharacter via the campaign's LobbyDO.
        // The DO's HTTP-dispatch endpoint accepts a server-originated intent
        // for this purpose; see lobby-do.ts § server-dispatch.
        const doId = c.env.LOBBY_DO.idFromName(campaignId!);
        const stub = c.env.LOBBY_DO.get(doId);
        const dispatchResp = await stub.fetch('https://internal/server-dispatch', {
          method: 'POST',
          body: JSON.stringify({
            type: 'SubmitCharacter',
            actor: { userId },
            source: 'server',
            payload: { characterId: id },
          }),
        });
        autoSubmitted = dispatchResp.ok;
      }

      return c.json({
        id,
        ownerId: userId,
        name: parsed.data.name,
        data: initialData,
        createdAt: now,
        updatedAt: now,
        autoSubmitted,
      });
    })
    .put('/:id', async (c) => {
      const userId = c.var.userId;
      const id = c.req.param('id');
      const body = await c.req.json();
      const parsed = UpdateCharacterRequestSchema.safeParse(body);
      if (!parsed.success) return c.json({ error: 'invalid_request' }, 400);
      const db = c.var.db;
      const existing = await db.select().from(characters).where(eq(characters.id, id)).get();
      if (!existing) return c.json({ error: 'not_found' }, 404);
      if (existing.ownerId !== userId) return c.json({ error: 'forbidden' }, 403);

      const newName = parsed.data.name ?? existing.name;
      const newData = parsed.data.data
        ? JSON.stringify(parsed.data.data)
        : existing.data;

      await db
        .update(characters)
        .set({ name: newName, data: newData, updatedAt: Date.now() })
        .where(eq(characters.id, id))
        .run();

      const updated = await db.select().from(characters).where(eq(characters.id, id)).get();
      return c.json({
        id: updated!.id,
        ownerId: updated!.ownerId,
        name: updated!.name,
        data: CharacterSchema.parse(JSON.parse(updated!.data)),
        createdAt: updated!.createdAt,
        updatedAt: updated!.updatedAt,
      });
    })
    .delete('/:id', async (c) => {
      const userId = c.var.userId;
      const id = c.req.param('id');
      const db = c.var.db;
      const row = await db.select().from(characters).where(eq(characters.id, id)).get();
      if (!row) return c.json({ error: 'not_found' }, 404);
      if (row.ownerId !== userId) return c.json({ error: 'forbidden' }, 403);
      await db.delete(characters).where(eq(characters.id, id)).run();
      return c.json({ ok: true });
    });
